## Technologies used for this project:


- Next.js: A React-based framework that enables server-side rendering and efficient client-side navigation, providing a seamless and performant web application.
- Tailwind CSS: A utility-first CSS framework that allows for rapid and maintainable styling of web applications, providing a highly customizable and low-level utility approach to building designs.
- React: A JavaScript library for building user interfaces, widely used in Next.js projects for creating reusable components and managing the UI state efficiently.
- JavaScript (ES6+): The programming language used for client-side scripting, enhancing the functionality and interactivity of the web application.
- Node.js: A JavaScript runtime that enables server-side execution of JavaScript code, crucial for server-side rendering in Next.js.
- Git: A version control system used for tracking changes in the project's source code, facilitating collaboration and code management.
- Webpack: A module bundler used in Next.js projects to bundle and optimize the JavaScript, CSS, and other assets for efficient delivery to the browser.
- Babel: A JavaScript compiler that allows developers to use the latest ECMAScript features by transforming them into an older version of JavaScript that is compatible with a broader range of browsers.
- Prisma: A modern database toolkit for Node.js and TypeScript that simplifies database access and management, providing type-safe database queries.
- Vercel: A cloud platform for deploying and hosting web applications, including Next.js projects, with features like serverless functions and automatic deployments.
- Stripe: A payment processing platform that enables businesses to accept online payments securely, often used for e-commerce functionalities.
- Mux: A platform for video streaming and hosting, providing APIs for video encoding, streaming, and playback.
- PlanetScale: A database platform for scaling MySQL databases, offering features like automatic sharding and seamless scaling for globally distributed applications.
